## Utilisation de AsciiGuessr

Afin d'utiliser le projet, il suffit de taper les commandes suivantes dans un terminal :

**Compilation des fichiers présents dans 'src' et création des fichiers '.class' dans 'classes'**

```
./compile.sh
```

**Lancement du jeu**

```
./run.sh AsciiGuessr
```
